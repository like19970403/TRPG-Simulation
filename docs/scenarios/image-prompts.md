# Crimson Manor - Image Generation Prompts

Style guide: All images should be **dark Victorian gothic, oil painting style, moody atmospheric lighting, cinematic composition**. Aspect ratio: **16:9 landscape** for scenes, **1:1 square** for items and NPCs.

Copy each prompt into ChatGPT and download the generated image. Save with the filename indicated.

## Workflow

1. Open ChatGPT, paste each prompt one by one
2. Download each generated image
3. Rename to the filename indicated (e.g., `scene_arrival.png`)
4. Place all images in: `web/public/scenarios/crimson-manor/`
5. After all images are ready, the scenario JSON will be updated with image paths
